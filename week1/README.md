JavaScript 是所有现代浏览器以及 HTML5 中的默认脚本语言,在html中插入脚本语言需要放在<scrip></scrip>之间。
脚本可被放置在 HTML 页面的 <body>·<head> 部分中，也可以把脚本保存到外部文件中。外部文件通常包含被多个网页使用的代码。
